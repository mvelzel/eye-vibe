# Eleventh wide horizon — second mixed batch freeze

## Why another mixed batch

The first batch did enough breadth work to justify one deeper engineering
attempt, but not enough to let that attempt absorb the search. The
ciphertext-only GAK orphan is the only new primitive that passed its exact
calibration. It receives one scaling experiment beside five unrelated audits.

Several tempting labels are already partly answered:

- direct continuous-panel overlap is exactly zero, and fixed-base continuous
  deck models are negative;
- the 26-column mutual-information effect failed family-held-out prediction;
- literal 5/42/83/101 Lua/XML/text tables and the procedural-wand 83-of-101
  threshold have already been inventoried;
- generic reconvergence and graph quotients have already been tested.

The second batch therefore asks narrower questions rather than rerunning those
families.

## Frozen batch

| Lane | Exact first test | Promotion gate |
|---|---|---|
| **A. Symbolic GAK orphan solver** | Encode unknown plaintext, unknown permutation operations, identity reset, distinct non-top source positions, and observed top cards in SMT-LIB. Reproduce the four-card parent/orphan pair and cross-check every decided tiny case against the enumeration oracle. | Reach a planted double-free instance at deck size at least 10 and decide at least one nontrivial UNSAT instance beyond deck size 4 before any Eye prefix is submitted. |
| **B. Composite-action collision calibration** | Generate planted sparse-XGAK pairs that diverge and later emit equal cards. | **C. Reset-regime identifiability audit** | Collect all existing reset, marker-reset, row-reset, continuation, and worldline results into one model matrix. Identify whether any label-invariant discriminator remains untested. | Implement a new regime classifier only if it distinguishes planted reset/continuation fixtures and is not equivalent to the already-zero suffix/prefix overlap or fixed-base scans. |
| **D. Renderer counterfactual audit** | Re-express the retained position-synchrony and recent change-point claims under every rewrap width 20–32. Separate absolute glyph index from row-end distance. | A statistic must select width 26 in controls and predict a held-out prefix family; the old in-sample column MI cannot be reused as validation. |
| **E. Executable later-clue interface** | Extend the installed-WAK audit from literal table size to executable arithmetic signatures: modulo/range 83 or 101, nine-way selectors, 42-pair loops, and functions that feed their result into a state walk or lookup. | A candidate must have data flow, not merely a literal, and must predict one Eye quantity not used in selecting the candidate. |
| **F. Practice transfer backstop** | Revisit unsolved practice ciphers 3 and 4 only for a method that would materially constrain lanes A–E: unknown-operation inference, reset detection, or route/interface recovery. | The method must solve or predict held-out practice data before transfer to Eyes. |

## Stop rules

- Lane A stops at `unknown` or below the planted size-10 gate; SAT on an
  underconstrained prefix is not progress.
- Lane B stops if true operation collisions are not identifiable from emitted
  equality alone on planted fixtures.
- Lane C and D may close by showing the proposed test is already subsumed by a
  committed negative.
- Lane E reports the complete search grammar and all hits; clue-named paths are
  not privileged after inspection.
- Lane F does not reopen broad brute-force searches already closed for practice
  cipher 4.
